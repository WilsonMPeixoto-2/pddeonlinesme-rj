# Skill: document-template-engine

Use para templates oficiais, Excel, DOCX, PDF, preenchimento de celulas, formulas, protecoes e geracao documental.

Regras:

- Confirmar template e mapa de celulas antes de implementar.
- Preservar formulas, layout, bordas, mesclagens e abas do arquivo oficial.
- Nao alterar template oficial sem revisao humana.
- Para Demonstrativo Basico Individual, preencher a aba `MEMORIA` diretamente.
- Nao depender da aba `BASE` nem de `XLOOKUP` para o arquivo individual.
