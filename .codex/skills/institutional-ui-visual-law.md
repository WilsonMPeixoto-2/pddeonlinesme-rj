# Skill: institutional-ui-visual-law

Use para polimento institucional, acessibilidade visual, consistencia de Light/Dark Mode e linguagem visual do PDDE Online 2026.

Regras:

- Preservar Light Mode e Dark Mode institucionais.
- Evitar redesign global sem decisao explicita.
- Nao alterar contrato de dados durante polimento visual.
- Validar responsividade quando houver mudanca de tela.
- Registrar mudancas visuais relevantes em handoff ou changelog apropriado.
