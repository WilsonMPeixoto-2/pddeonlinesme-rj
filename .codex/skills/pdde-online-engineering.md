# Skill: pdde-online-engineering

Use para tarefas de engenharia geral no PDDE Online 2026.

Antes de agir:

1. Ler `AGENTS.md`.
2. Ler `.continuity/current-state.json`.
3. Ler `docs/HANDOFF.md`.
4. Ler `docs/DECISIONS.md`.
5. Confirmar arquivos permitidos e proibidos.

Regras:

- Preservar TypeScript strict.
- Preferir PR pequeno e verificavel.
- Nao misturar governanca, UI, Supabase e documentos oficiais no mesmo PR sem autorizacao.
- Atualizar continuidade e handoff ao final.
