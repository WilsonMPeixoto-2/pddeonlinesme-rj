# Workflow: new-pr

Use para iniciar um PR pequeno no PDDE Online 2026.

1. Ler `AGENTS.md`, `.continuity/current-state.json`, `docs/HANDOFF.md`, `docs/DECISIONS.md`, `docs/ROADMAP_ADAPTIVE.md` e `docs/OPPORTUNITIES_BACKLOG.md`.
2. Confirmar branch base, branch atual e estado do worktree.
3. Declarar ferramenta lider, objetivo, arquivos permitidos, arquivos proibidos, validacao e ponto de handoff.
4. Criar branch com nome curto e escopo unico.
5. Alterar apenas os arquivos permitidos.
6. Rodar a menor validacao coerente com o escopo.
7. Atualizar continuidade e handoff.
8. Abrir PR com titulo, resumo, escopo negativo e validacoes executadas.

Regra: nao misturar PR documental com mudanca funcional.
